## 使用說明

	python jd.py -k 芒果

	三個參數：
	-d	保存圖片的路徑，默認爲fd.py文件所在文件夾
	-k	搜索關鍵詞
	-n  	下載商品的曬單圖個數，即n個商店的曬單圖
